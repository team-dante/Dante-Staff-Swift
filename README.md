# Dante Staff

## Getting Started
1. Open the terminal, go to the directory where you would like to store this project.
2. Type `git clone https://github.com/team-dante/Dante-Staff-Swift.git`
3. Navigate to project folder `cd Dante-Staff-Swift`
4. Then, open `Dante Staff.xcworkspace`; hit play at the top-left corner of Xcode
